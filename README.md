A fully automated V2Ray subscription aggregator. This repository collects, cleans, and sorts configurations from various high-quality public sources every 60 minutes.

**Features:**
- üîÑ **Auto-Updates:** Runs automatically every 60 minutes.
- ‚ö° **Optimized:** Configs are shuffled and split into small chunks (500 per file) for faster loading.
- üßπ **Clean:** Smart Base64 decoding and deduplication.
- üìÇ **Categorized:** Separate files for specific protocols (VMess, VLESS, etc.).

   - Just paste the link in the subscription field.
